# Note Effects

Note Effects (Raining Effect) displays a falling note animation when keys are pressed.

## Enabling Note Effects

1. Go to the **Settings** tab.
2. Find the **Note Effects** section.
3. Turn on the **Enable Note Effects** toggle.

## Settings Options

### Color

Set the color of notes. Use the color picker to select your desired color.

### Opacity

Adjust the opacity of notes.

- **0%**: Fully transparent
- **100%**: Fully opaque

### Rounding

Set the roundness of note corners.

- **0px**: Sharp corners
- **Higher values**: More rounded corners

### Speed

Adjust the falling speed of notes.

- **Lower values**: Falls slowly
- **Higher values**: Falls quickly

### Height

Set the maximum height that notes fall.

### Reverse

Reverse the direction notes fall.

- **Disabled**: Falls from top to bottom
- **Enabled**: Rises from bottom to top

## Usage Tips

<Callout type="info">
  Note effects are displayed continuously while keys are held. Useful for
  visualizing long notes in rhythm games.
</Callout>

- Matching note effect colors with in-game note colors creates a more consistent visual experience.
- Setting the speed similar to the game's note speed creates a natural feel.
