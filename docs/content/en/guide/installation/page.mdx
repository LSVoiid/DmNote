# Installation & Running

## Download

DM Note can be downloaded from GitHub Releases.

1. Download the latest version from the [GitHub Releases page](https://github.com/lee-sihun/DmNote/releases).
2. Extract the downloaded ZIP file to your desired location.
3. Run the `DM Note.exe` file.

<Callout type="info">
  **No installation process required.** It's a portable format that you can run
  immediately after extraction.
</Callout>

## System Requirements

- **Operating System**: Windows 10/11 (64-bit)
- **Runtime**: WebView2 Runtime (pre-installed on Windows 10 and above)

## First Run

When you first run the program, two windows will appear.

1. **Main Window**: Window for managing settings and key mapping
2. **Overlay Window**: Window where actual key inputs are displayed

<Callout type="warning">
  **Security warning may appear.** If Windows SmartScreen shows "Windows
  protected your PC" message, click "More info" and then select "Run anyway".
</Callout>

## Settings File Location

Program settings are automatically saved to the following path.

```
%appdata%/com.dmnote.desktop/store.json
```

If you want to back up all settings, we recommend making a copy of this file.
