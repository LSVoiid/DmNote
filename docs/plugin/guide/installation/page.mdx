# 설치 및 실행

## 다운로드

DM Note는 GitHub Releases에서 다운로드할 수 있습니다.

1. [GitHub Releases 페이지](https://github.com/lee-sihun/DmNote/releases)에서 최신 버전을 다운로드합니다.
2. 다운로드한 ZIP 파일의 압축을 원하는 위치에 해제합니다.
3. `DM Note.exe` 파일을 실행합니다.

<Callout type="info">
  **별도 설치 과정이 필요 없습니다.** 압축을 해제한 후 바로 실행할 수 있는
  포터블 형태입니다.
</Callout>

## 시스템 요구사항

- **운영체제**: Windows 10/11 (64-bit)
- **런타임**: WebView2 Runtime (Windows 10 이상에서는 기본 설치됨)

## 첫 실행

프로그램을 처음 실행하면 두 개의 창이 나타납니다:

1. **메인 창**: 설정과 키 매핑을 관리하는 창
2. **오버레이 창**: 실제로 키 입력이 표시되는 투명한 창

<Callout type="warning">
  **보안 경고가 나타날 수 있습니다.** Windows SmartScreen에서 "Windows가 PC를
  보호했습니다" 메시지가 나타나면, "추가 정보"를 클릭한 후 "실행"을 선택하세요.
</Callout>

## 설정 파일 위치

프로그램 설정은 다음 경로에 저장됩니다:

```
%appdata%/com.dmnote.desktop/store.json
```

설정을 초기화하려면 이 파일을 삭제하면 됩니다.

## 업데이트

새 버전이 출시되면:

1. 기존 폴더를 백업합니다.
2. 새 버전을 다운로드하여 압축을 해제합니다.
3. 기존 설정을 유지하려면 `%appdata%/com.dmnote.desktop` 폴더의 내용을 그대로 두세요.
