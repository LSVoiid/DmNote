# 노트 이펙트

노트 이펙트(Raining Effect)는 키를 누를 때 떨어지는 노트 애니메이션을 표시하는 기능입니다.

## 노트 이펙트 활성화

1. **Settings** 탭으로 이동합니다.
2. **노트 이펙트** 섹션을 찾습니다.
3. **노트 이펙트 활성화** 토글을 켭니다.

## 설정 옵션

### 색상

노트의 색상을 설정합니다. 컬러 피커를 사용하여 원하는 색상을 선택하세요.

### 투명도

노트의 투명도를 조절합니다.

- **0%**: 완전히 투명
- **100%**: 완전히 불투명

### 라운딩

노트 모서리의 둥근 정도를 설정합니다.

- **0px**: 직각 모서리
- **높은 값**: 더 둥근 모서리

### 속도

노트가 떨어지는 속도를 조절합니다.

- **낮은 값**: 느리게 떨어짐
- **높은 값**: 빠르게 떨어짐

### 높이

노트가 떨어지는 최대 높이를 설정합니다.

- **활성화**: 아래에서 위로 올라감

## 사용 팁

<Callout type="info">
  노트 이펙트는 키를 누르고 있는 동안 계속 표시됩니다. 리듬게임에서 롱노트를
  시각화하는 데 유용합니다.
</Callout>

- 리듬게임의 노트 색상과 맞추면 더 일관된 시각적 효과를 얻을 수 있습니다.
- 속도 설정을 게임의 노트 속도와 비슷하게 맞추면 자연스럽습니다.
