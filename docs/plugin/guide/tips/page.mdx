# 팁 & 참고사항

DM Note를 더 효과적으로 사용하기 위한 팁과 알아두면 좋은 사항들을 정리했습니다.

## 게임과 함께 사용하기

### 전체화면 모드 문제

일부 게임의 전체화면 모드에서는 오버레이가 정상 동작하지 않을 수 있습니다.

**해결 방법:**

- 게임을 **테두리 없는 창 모드**로 실행하세요.
- 또는 게임 설정에서 **창 모드**를 선택하세요.

### 오버레이 배치 팁

게임 화면 위에 오버레이를 배치하려면:

1. **항상 위에 표시** 옵션을 활성화합니다.
2. 오버레이를 원하는 위치로 이동합니다.
3. **오버레이 창 고정** 옵션을 활성화합니다.

<Callout type="info">
  창 고정을 활성화하면 실수로 오버레이를 이동하는 것을 방지할 수 있습니다.
</Callout>

## OBS 설정

### 투명 배경으로 캡쳐하기

OBS Studio에서 크로마키 없이 투명 배경을 유지하려면:

1. **소스 추가** > **윈도우 캡쳐**를 선택합니다.
2. **창**에서 `DM Note` 오버레이 창을 선택합니다.
3. **캡쳐 방법**을 `Windows 10 (1903 이상)`으로 설정합니다.
4. **클라이언트 영역 캡쳐**를 체크합니다.

### 게임과 오버레이 함께 캡쳐하기

게임과 오버레이를 한 번에 캡쳐하려면:

- **게임 캡쳐**로 게임을 추가합니다.
- **윈도우 캡쳐**로 오버레이를 추가합니다.
- 오버레이 소스를 게임 소스 위에 배치합니다.

## 성능 최적화

### 그래픽 문제 해결

오버레이에 깜빡임이나 렌더링 문제가 있다면:

1. **Settings** > **그래픽 설정**으로 이동합니다.
2. **렌더링 옵션**을 다른 값으로 변경해 봅니다.
   - Direct3D 11 → Direct3D 9
   - Direct3D 9 → OpenGL

### 리소스 사용 줄이기

- 사용하지 않는 노트 이펙트는 비활성화하세요.
- 필요 없는 플러그인은 비활성화하세요.

## 문제 해결

### 키 입력이 감지되지 않는 경우

1. 프로그램을 **관리자 권한**으로 실행해 보세요.
2. 백신 소프트웨어가 프로그램을 차단하고 있는지 확인하세요.
3. 키 매핑이 올바르게 설정되어 있는지 확인하세요.

### 설정이 저장되지 않는 경우

1. `%appdata%/com.dmnote.desktop` 폴더에 쓰기 권한이 있는지 확인하세요.
2. 폴더가 없다면 프로그램을 다시 실행하여 자동으로 생성되도록 합니다.

### 프로그램이 시작되지 않는 경우

1. **WebView2 Runtime**이 설치되어 있는지 확인하세요.
2. Windows 10/11에서는 기본 설치되어 있지만, 없다면 [Microsoft 공식 사이트](https://developer.microsoft.com/microsoft-edge/webview2/)에서 다운로드할 수 있습니다.

## 커스텀 CSS 팁

### 실시간 반영

커스텀 CSS는 실시간으로 반영되지 않습니다. CSS 파일을 수정한 후:

1. 설정에서 CSS 파일을 다시 선택하거나
2. 프로그램을 재시작하세요.

### CSS 디버깅

`Ctrl+Shift+I`로 개발자 도구를 열어 요소를 검사하고 CSS를 테스트할 수 있습니다.

## 백업

중요한 설정은 정기적으로 백업하세요:

```
%appdata%/com.dmnote.desktop/
```

이 폴더 전체를 복사하면 모든 설정, 프리셋, 플러그인 데이터가 백업됩니다.
